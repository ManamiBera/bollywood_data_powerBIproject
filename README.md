## Project Title | Bollywood Movie Analytics Dashboard

## Project Overview

This project presents an interactive analytics dashboard built using Microsoft Power BI Desktop to analyze Bollywood movie data and uncover industry trends.

The dashboard focuses on understanding:

•Box office performance

•Movie ratings and audience reception

•Genre trends over time

•Top actors and successful movies

The objective of this project is to demonstrate how data visualization and business intelligence tools can be used to generate meaningful insights from entertainment industry data.

## Problem Statement

The Bollywood industry produces hundreds of movies each year, making it difficult to identify patterns in performance, popularity, and success factors.

This project aims to analyze movie data and present insights in a clear and interactive format.

## Dashboard Features

•Interactive filters for year, genre, and movies

•KPI metrics for overall industry performance

•Visual charts for trend analysis

•Actor and movie performance insights

•Genre popularity visualization

## Tools & Technologies Used

•Power BI

•Excel

•Data Cleaning

•Data Modeling

•DAX

## Dataset

The dataset includes:

•Movie titles

•Release year

•Box office revenue

•Ratings

•Genres

•Cast information

## Key Insights

Some insights identified from the dashboard include:

•Certain genres consistently perform better at the box office

•A small number of movies generate a large portion of revenue

•Movie ratings often influence box office success

•Industry trends change across years
